# VoiceClaw — Plumber Landing Page Concept

A concept landing page for VoiceClaw, an AI receptionist built for emergency plumbers.

This is a design concept for the plumber ICP (Ideal Customer Profile) — a tradesperson-facing landing page with craft-respect positioning, the "2am call you can't answer from a crawl space" narrative, and a voice-first setup CTA.

## View it

- **Locally**: open `index.html` in any modern browser
- **GitHub Pages** (if enabled in repo settings): `https://ianpilon.github.io/voiceclaw-plumber-landing-concept/`

## What's in here

A single self-contained HTML file using TailwindCSS (via CDN) and Inter (Google Fonts). No build step, no dependencies, no install.

## Design choices

- **Palette**: deep slate/navy with orange accents (Carhartt / construction / emergency-trade vibe) rather than the clinical green of a physiotherapy-focused concept.
- **Copy**: craft-respect opener honoring the plumber's identity before any product claim. Plumber-native sensory language throughout (crawl space, 2am burst pipe, slab leak, water heater replacement, Ferguson counter, handyman with a pressure washer).
- **Problem framing**: three pain cards naming specific enemies — the 2am call lost to Roto-Rooter's overnight service, the Angi lead-fee treadmill and Benjamin Franklin national ads, and the AI-discoverability lineup the plumber isn't in.
- **Comparison table**: direct against Ruby Receptionist, Rosie/Goodcall, and Angi Pro Leads (named alternatives plumbers have usually tried).
- **Math**: plumber-native dollar amounts ($2K water heater installs, $75 Angi leads, $47K annual impact).
- **CTAs**: primary "Try the voice setup now" (voice onboarding experience) with clear context — "2 minutes, no credit card, you talk, we build."

## Designed as the first in a series

Planned ICP-specific landing page concepts across the seven trades:

- **Plumber** (this page)
- Painter (via Sherwin-Williams channel)
- HVAC technician
- Roofer
- Electrician
- Pest control
- Locksmith

Each page reuses the underlying structure but swaps the sensory vocabulary, competitor names, dollar-amount anchors, and trade-specific enemy list.
